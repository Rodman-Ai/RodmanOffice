LeoCRM

AI lead-generation CRM that lives in your Google Workspace.

Sign in with Google. LeoCRM provisions a spreadsheet and a Drive folder in your account — that's the entire database. Claude generates personalized outreach, you review, and Gmail sends it. Replies auto-bump pipeline stages. Sequences run on a delay. No vendor lock-in: your data stays in your Sheet.

Sheets-as-database

Contacts, leads, companies, campaigns, sequences, tasks, and full email history live as tabs in LeoCRM Workspace on your Drive.

Multi-step AI sequences

Build N-step cadences with per-step instructions and delays. Each send is freshly personalized by Claude. Replies auto-stop the sequence.

AI lead scoring + A/B subjects

One-click 0–100 score with a one-line rationale. Generate two subject variants per email and track which converts.

Activity timeline + tasks

Every send, reply, stage change, score, task, and form submission is logged per contact. Open tasks surface on the dashboard.

Public lead-capture forms

Spin up a /f/<slug> page that creates a contact, opens a lead, and optionally enrolls into a sequence on submit.

Desktop + mobile (PWA)

Responsive shell with a desktop sidebar and mobile bottom-nav. Add to Home Screen on iOS/Android for an app-like feel.

Source on GitHub Self-host setup
You're on the GitHub Pages landing page. The full app needs a server (NextAuth, Google API calls, AI generation), so it can't run on Pages. Deploy the app to Vercel, Cloudflare Pages, or Netlify with the env vars in .env.example — the .github/workflows/deploy.yml workflow ships to Vercel automatically once you add the secrets.
Built with Next.js 14, Tailwind, NextAuth, googleapis, and the Anthropic SDK.
